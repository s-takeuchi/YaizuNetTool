stk_ip_dlg


This library provides a dialog box for registring IP addresses.
Notice:
- This program can manage up to 11 network adapters. In case of more than 11 adapters exist, first 11 found adapters are managed.
- If specified IP address for addition has already been registered in the target adapter, registering process of the address is ignored.
- If specified IP address for removal has already been removed from the target adapter, removing process of the address is ignored.

本ウィザードは，対象となるオペレーティングシステムの種類に応じて処理を切り替えている。
Windows 2000 ProfessionalIP Helper APIを用いてIPアドレスの追加／削除を行う。 このため，本ウィザードで行った変更は，オペレーティングシステムをシャットダウンするまで有効となる。 オペレーティングシステム再起動後，本ウィザードで行った変更はもとにもどる。
Windows XP Professional
Windows Vista Business		netshコマンドを用いてIPアドレスの 追加／削除を行う。オペレーティングシステム再起動後も，本ウィザードで行った変更は維持される。

Function __declspec(dllimport) void ShowStkIpDlg(HWND)
Description Shows the dialog box for registring IP addresses.
In HWND: Window handler. Specify NULL if no parent exists.
Out VOID
