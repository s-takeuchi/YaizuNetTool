stk_about


This library provides the common dialog box for showing 'About' information.

Function void SetAboutTitle(TCHAR*)
Description This function sets the title of the program which uses stk_about.
The title can be specified in 255 characters.
In TCHAR*: Specify the title of the program.
Out VOID

Function void AddHistoryItem(WORD, TCHAR*, TCHAR*, TCHAR*)
Description This function adds the update history into stk_about module.
User program can add 128 update histories as max.
Change ID of the update history can be specified in 4 characters.
Version of the update history can be specified in 9 characters.
Description of the update history can be specified in 255 characters.
In WORD: Update type (0:Bug fix, 1:Enhancement)
TCHAR*: Change ID
TCHAR*: Version
TCHAR*: Description
Out VOID

Function void ShowAboutDialog(HWND)
Description This function shows the about dialog.
In HWND: Window handle of the program which stk_about uses. Specify NULL if no parent window exists.
Out VOID

Function void ClearHistoryItem()
Description This function clears all of history items.
In
Out VOID

Function void SetStkAboutIconAbout(int)
void SetStkAboutIconBug(int)
void SetStkAboutIconEnhance(int)
Description This functions set the icon.
SetStkAboutIconAbout : Set system icon
SetStkAboutIconBug : Set icon for 'Bug fix'
SetStkAboutIconEnhance : Set icon for 'Enhancement'
In int: Resource ID for the icon
Out VOID
