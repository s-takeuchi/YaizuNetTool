2. Installation / Uninstallation / Program launch

2.1 Installation

Follow the steps outlined below for installation.
(1) Execute stkfw.msi
(2) Follow the installer dialog.

* Installation must be performed by a logged-in user with administrator rights.
* The installer automatically registers the program as a firewall exception.
* If an older version of StkFw is already installed, the installer program will uninstall it before installing the new version.

2.2 Uninstall

Follow the steps outlined below for uninstallation.
(1) Execute stkfw.msi and select Remove StkFw, or launch [Add/Remove Programs] from the Control Panel and select StkFw for removal.
(2) Follow the uninstaller dialog.

* Uninstallation must be performed by a logged-in user with administrator rights.
* The uninstaller automatically removes firewall exceptions for this program.

2.3 Program launch

Upon installation, a StkFw shortcut is created in the Start menu.
The main StkFw console will be launched when this shortcut is selected.
If the sample shortcut under [StkFw]-[Samples] on the Start menu is clicked, the main StkFw console will also be launched with associated sample data.
