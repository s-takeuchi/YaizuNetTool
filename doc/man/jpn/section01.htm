1. StkFwについて

1.1 StkFw概要

StkFwは，「複数のソフトウェア／ハードウェアがIPネットワーク上で連携して動作する環境」を シミュレートするテスト支援ソフトウェアです。

ネットワーク上の複雑で高価な要素（ソフトウェアやハードウェア）と相互接続／連携して動作するソフトウェアの開発で， 各要素の振る舞いをテストプログラム化することにより開発対象ソフトウェアのテスト工数を削減する手法は一般に良く知られています。
しかし，テストプログラムの振る舞い（他ネットワーク要素との遣り取り）が複雑な場合， テストプログラム開発のための初期投資費用が高くなる場合があります。

StkFwは，優れたユーザーインターフェースにより，IPネットワーク上でのデータの送受信を簡易且つ詳細に制御できます。
これにより実際にテストプログラムを開発しなくても，テストプログラムと同等の振る舞いをStkFwで実現することができ， ソフトウェアのテストに関わる工数を低減することができます。

StkFwは以下の特徴を持ちます。

  • データの受信，加工，送信をアイコンとアイコン間の関連線で表現したデータフローとして定義することができます。 これによりネットワーク上の複雑なやり取りを簡易且つ即座に作成することができます。
  • データの加工では，様々な加工条件や加工種別を設定することができ，テストプログラムの振る舞いを詳細に模倣することができます。
  • ループバックアダプタ追加機能や複数IPアドレス追加機能等， ネットワーク上の複数の要素の振る舞いをローカルマシン上で実現するための機能を備えています。

1.2 システム要件

StkFwは以下の環境で動作します。
  • 動作に必要なCPU:
    1GHz以上のクロック周波数で動作するCPU

  • 動作に必要なメモリ容量:
    512MBytes以上

  • 導入に必要な空きディスク容量:
    20MBytes以上

  • サポートするオペレーティングシステム／エディション: ※1
    Windows 2000 Professional SP4 ※2
    Windows XP Professional SP3
    Windows XP Professional x64 SP2
    Windows Vista Business/Ultimate SP2
    Windows 7 Professional/Ultimate SP1
    Windows 8 Pro
    Windows Server 2003 Standard SP2
    Windows Server 2003 R2 Standard SP2
    Windows Server 2008 Standard SP2
    Windows Server 2008 R2 Standard SP1
    Windows Server 2012 Standard
※1 オペレーティングシステムのデフォルトの文字サイズを変更した場合，動作に支障はないものの， ダイアログ等のレイアウトが崩れます。

※2 Windows 2000 SP4 用の更新プログラム ロールアップ 1が適用されている必要があります。

1.3 使用許諾条件

StkFw（以下本ソフトウェアと表記）はフリーウェアです。利用者は以下の条件のもと本ソフトウェアを無償で利用することができます。

- 本ソフトウェアを（利用者本人と製作者を除く）第三者に再配布する場合， 本ソフトウェアに含まれる全てのファイルをパッケージ形式も含め無改変で配布しなければなりません。
